M1 — Vectors

|A| = sqrt(Ax^2 + Ay^2)
|A| = sqrt(Ax^2 + Ay^2 + Az^2)
theta = arctan(Ay / Ax)
u-hat = A / |A|

M2 — Vector Operations

A.B = |A||B|cos(theta) = AxBx+AyBy+AzBz
A x B : right-hand rule, result is vector
A.B = 0 means perpendicular

M3 — Kinematics 1D

vf = v0 + at
Dx = (1/2)(v0+vf)t
Dx = v0*t + (1/2)a*t^2
vf^2 = v0^2 + 2a*Dx
g = -9.8 m/s^2 (up is positive)

M4 — Projectile & Circular

x = v0*cos(th) * t
y = v0*sin(th)*t - (1/2)g*t^2
R = v0^2 * sin(2th) / g (max at 45deg)
ac = v^2/r, Fc = m*v^2/r
f = 1/T, v = 2*pi*r/T

M5–M6 — Newton's Laws

F = ma, W = mg
Wx = W*sin(th), N = W*cos(th)
f = mu*N (mu_s > mu_k)
Atwood: a = (m1-m2)g/(m1+m2)
Pulley block: a = m2*g/(m1+m2)
Elevator up: T = m(g+a)

M7 — Work & Energy

W = F*d*cos(th) [Joules]
KE = (1/2)*m*v^2
PE = m*g*h
Wnet = DeltaKE (work-energy theorem)
P = W/t [Watts]

M8 — Impulse & Momentum

p = m*v
I = Dp = m(vf - v0)
Elastic: both p and KE conserved
Inelastic: m1v1 + m2v2 = (m1+m2)vf

M9 — Equilibrium & Torque

tau = r*F*sin(th) [N.m]
1st cond: Sum_F = 0
2nd cond: Sum_torque = 0
tau max at th=90deg, zero at th=0deg
